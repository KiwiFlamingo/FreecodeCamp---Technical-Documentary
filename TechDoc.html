Introduction

HTML stands for HyperText Markup Language (HTML) and it is a standard computer languaged used to structure and create websites. A markup language means it requires tags to communicate on what to display. You may add styling to the site with CSS, and interactivity to the site with JavaScript.

When using this language, it is best to remember tags and its attributes - while they are not case sensitive - are best to write in lowercase for effective coding. Some tags may not need a closing tag so it is convinient for you to memorize which tags do and dont close. While placing tags inside tags, remember to follow nesting rules: "Close tags in the reversed order they were opened."

Important Tags

  • <html>: Defines an HTML document.

  • <head>: Contains information about your HTML document. It will make it easier to find. It is also good for connecting separate styling and interactive files.

  • <body>: Contains the content of the document.

  • <!DOCTYPE html>: Tells browsers which version of HTML the document is written in (HTML5).
